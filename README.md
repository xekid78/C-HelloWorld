# CsHelloWorld
HelloWorldと表示（.NET Core）

## 処理
コンソールアプリとして画面にHelloWorldと表示

## コード
```
using System;

namespace Hello
{
    class Hello
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Hello World");
        }
    }
}
```

## 出力結果  
```
Hello World
```
